What is an audiodrama?

An audiodrama is a scripted, voice-acted, fiction podcast, similar to a TV show without the visual aspect. Audiodramas are a relatively niche medium dominated by small creators. This website is a record of audiodramas I've lisened to, designed to be easily searched.
